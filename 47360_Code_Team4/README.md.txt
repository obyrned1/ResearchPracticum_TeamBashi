Dublin Bashi is a web application that can route and predict times for Dublin Bus journeys. 
The application gives relevant waiting and timetable times to the user.
It is optimised for mobile devices.
The application is currently running (24/08/2018) on www.dublinbashi.com

To deploy this application:

1. Open the Bashi folder in the command prompt
2. run: pip install -r requirements.txt
3. In the Bashi folder, run: python manage.py runserver
4. The application will be running on your localhost, port 8000

Enjoy!
